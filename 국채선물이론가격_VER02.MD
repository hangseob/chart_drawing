# [별표 9] 국채선물이론가격 (VER02)

<개정 2010.10.20, 2011.1.27, 2011.4.21., 2017.3.13., 2023.9.22., 2024.2.5., 2025.5.29>

(제55조제1항 관련)

## 1. 이론가격 산출식
국채선물의 이론가격은 다음 각 목의 구분에 따라 산출한 수치(소수점 셋째자리에서 반올림)로 한다.

*   **가. 3년국채선물**: 3년국채선물 이론가격 = $\sum_{i=1}^{6} \frac{\frac{5}{2}}{(1+\frac{r}{2})^i} + \frac{100}{(1+\frac{r}{2})^6}$
*   **나. 5년국채선물**: 5년국채선물 이론가격 = $\sum_{i=1}^{10} \frac{\frac{5}{2}}{(1+\frac{r}{2})^i} + \frac{100}{(1+\frac{r}{2})^{10}}$
*   **다. 10년국채선물**: 10년국채선물 이론가격 = $\sum_{i=1}^{20} \frac{\frac{5}{2}}{(1+\frac{r}{2})^i} + \frac{100}{(1+\frac{r}{2})^{20}}$
*   **라. 30년국채선물**: 30년국채선물 이론가격 = $\sum_{i=1}^{60} \frac{\frac{5}{2}}{(1+\frac{r}{2})^i} + \frac{100}{(1+\frac{r}{2})^{60}}$

---

## 2. 평균선도수익률($r$) 산출방법
상기 산식에서 “$r$”은 최종결제기준채권의 평균선도수익률을 말하며, 그 산출방법은 다음 각 목의 순서에 따른다.

### 가. 개별기준채권의 시장가격 산출
한국금융투자협회가 공시하는 개별기준채권의 유통수익률($r_1$)을 적용하여 다음의 산식에 따라 개별기준채권의 시장가격을 산출한다.

$$\text{시장가격} = \frac{1}{\left(1 + \frac{r_1}{2} \times \frac{d_1}{t_1}\right)} \times \left[ \sum_{i=0}^{n-1} \frac{\frac{c}{2}}{\left(1 + \frac{r_1}{2}\right)^i} + \frac{100}{\left(1 + \frac{r_1}{2}\right)^{n-1}} \right] \rightarrow \text{산식}(1)$$

*   $r_1$ : 개별기준채권의 유통수익률
*   $d_1$ : 산출일부터 차기 이자지급일까지의 일수
*   $t_1$ : 직전 이자지급일부터 차기 이자지급일까지의 일수
*   $n$ : 잔여 이자지급회수
*   $c$ : 개별기준채권의 표면금리 $\times$ 100

### 나. 개별기준채권의 선도가격 산출
가목에 따라 산출한 개별기준채권의 시장가격을 적용하여 다음 산식에 따라 개별기준채권의 선도가격을 산출한다.

$$\text{선도가격} = (S - I) \times \left(1 + r^* \times \frac{t}{365}\right)$$

*   $S$ : 개별기준채권의 시장가격
*   $I$ : 이자락금액의 현재가치 = $\frac{6\text{개월간의 이자지급액}}{(1 + r_2 \times \frac{d_2}{365})}$
*   $r_2$ : 이자락 해당 시 산출일부터 차기 이자지급일까지의 적용금리 (콜금리, CD수익률, 통안증권수익률을 선형보간하여 산출)
*   $d_2$ : 이자락 해당 시 이자산출일부터 차기 이자지급일까지의 일수
*   $r^*$ : 산출일부터 최종거래일까지의 적용금리
*   $t$ : 산출일부터 최종거래일까지의 산출잔존기간의 일수

### 다. 선도수익률 산출
가목의 산식을 적용하여 개별기준채권의 선도가격과 일치하는 선도수익률을 산출한다.

### 라. 최종 평균선도수익률 결정
개별기준채권의 선도수익률을 단순평균한 후 소수점 넷째자리에서 반올림하여 최종결제기준채권의 평균선도수익률을 산출한다.

## 비고
1. 야간거래시간에 이론가격을 산출하는 경우 “전일”은 “해당 거래일의 정규거래시간이 속하는 날의 전일”로 보고, “산출일”은 “해당 거래일의 정규거래시간이 속하는 날”로 본다.
2. 그 밖에 야간거래시간에 이론가격을 산출하는 경우 기간의 계산은 제1호를 준용한다.
